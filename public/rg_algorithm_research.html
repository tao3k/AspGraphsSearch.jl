ASP rg algorithm research

This notebook studies ripgrep as an algorithmic stage in the ASP search flow. The focus is pattern selectivity, path and language filtering, repeated scan pressure, and which rg searches should be replaced by provider-owned structural or graph queries.

Current artifacts distinguish raw rg shell commands from ASP provider search pattern commands. Raw rg can be zero while provider pattern-search still provides evidence for rg-like algorithm research.
