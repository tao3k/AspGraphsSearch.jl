#+TITLE: 10.01 ASP Artifacts Data Science Research
#+AUTHOR: AspGraphsSearch.jl

* One-sentence summary
AspGraphsSearch.jl studies ASP search/query artifacts as a typed action-flow dataset: JSON packets, provider command receipts, weak time phases, language/query taxonomy, graph algorithms, and instrumentation gaps are separated before notebook publication.

* Problem
The initial notebook counted command shapes and rendered a graph, but it did not meet the standard of a modular data-science algorithm study. A useful research surface must first define what the dataset contains, which facts are strong or weak evidence, how agent-triggered search actions can be reconstructed, and which algorithm experiments can produce actionable improvements.
